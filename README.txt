# Student Grade Tracker (Java Swing)

## Requirements
- Java Development Kit (JDK) 8 or later
- No external libraries needed (uses only built-in Java Swing and AWT).

## How to Run
1. Open terminal/command prompt.
2. Navigate to the folder containing `StudentGradeTrackerGUI.java`.
3. Compile the program:
   ```
   javac StudentGradeTrackerGUI.java
   ```
4. Run the program:
   ```
   java StudentGradeTrackerGUI
   ```

## Features
- Add student with name and grade (0-100).
- Displays student list in a table with grade category (A, B, C, Fail).
- Shows statistics: average, highest, lowest.
- Visualizes grades in a bar chart with category colors.
- Saves all data into `students.txt` automatically.
- Loads saved data when program starts.
